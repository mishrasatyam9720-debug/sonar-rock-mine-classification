# Rock vs Mine Prediction

## Overview
This is a Machine Learning project that predicts whether a sonar signal is reflected from a Rock or a Mine.

The model is trained on sonar data and uses classification techniques to identify the object based on the received signal patterns.

## Dataset
- Sonar Dataset
- Features: 60 numerical attributes
- Target:
  - R = Rock
  - M = Mine

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Google Colab

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Model Evaluation
6. Prediction System

## Results
The model successfully classifies sonar signals as Rock or Mine with good accuracy.
